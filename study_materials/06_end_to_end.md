# 06. End-to-End (E2E) 자율주행

> 인지→판단→제어를 **나누지 않고**, 센서(카메라)에서 조향까지 신경망 하나로.

## 행동 복제 (Behavioral Cloning) — P3
- 사람 주행 데이터(이미지 → 조향각)를 CNN이 그대로 모방(지도학습).
- **NVIDIA CNN**: 정규화→크롭→Conv 5층→FC 4층→조향각 1개.
- 데이터 증강: 좌/우 카메라 보정 오프셋, 좌우 반전, 밝기 변화.

## 모듈러 vs E2E
| | 모듈러 (P1~P11) | E2E (P3) |
|--|----------------|----------|
| 구조 | 인지/추적/계획/제어 분리 | 신경망 하나 |
| 해석성 | 높음(단계별 디버깅) | 낮음(블랙박스) |
| 데이터 | 단계별 라벨 | 주행 로그(이미지+조향) |
| 강건성 | 분포 밖에도 대응 | 본 적 없는 상황 취약 |
| 현업 | 양산차 주류 | 연구·일부(예: 일부 L2) |

## 발전 흐름
- 행동복제 → 강화학습(보상) → 모방+강화 혼합.
- 최신: 멀티모달(카메라+라이다) 트랜스포머, 월드모델 기반 E2E.

## 핵심 통찰
- E2E는 단순하지만 **"왜 그렇게 운전했는지" 설명이 어렵다** → 안전검증이 난제.
- 그래서 현재 양산은 모듈러가 주류, E2E는 빠르게 추격 중.

## 관련 프로젝트
P3(Behavioral Cloning). P1~P11(모듈러)과 비교하며 학습.
