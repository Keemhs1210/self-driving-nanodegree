# 03. 판단/위치추정 (Localization) — 파티클 필터

> "지금 내가 지도 위 정확히 어디에 있는가?" cm 단위로 추정.

## 몬테카를로 위치추정 (파티클 필터) — P8
```
1. init       GPS 근처에 N개 가설(파티클) 뿌림
2. predict    자전거 모델로 모든 파티클 이동 + 노이즈
3. update     각 파티클의 예상 관측 vs 실제 관측 → 가중치(다변량 가우시안)
4. resample   가중치 비례 재추출 → 그럴듯한 가설만 생존
```
반복하면 파티클 구름이 진짜 위치로 수렴.

## 핵심 수식
- **자전거 모델**(yaw_rate≠0):
  `x += v/ψ̇·(sin(ψ+ψ̇·dt)-sin ψ)`, `y += v/ψ̇·(cos ψ - cos(ψ+ψ̇·dt))`, `ψ += ψ̇·dt`
  (ψ̇≈0이면 직진 근사 — 0 나눗셈 분기 필수)
- **가중치**: 2D 다변량 가우시안 곱 (관측-랜드마크 거리).

## 왜 칼만이 아니라 파티클?
- 칼만은 단봉(가우시안) 가정. 위치는 **다봉**(여러 후보)일 수 있어 파티클이 자연스러움.

## 함정
- 좌표 변환(차량→지도): 회전+평행이동 순서 주의.
- 파티클 수 N: 정확도↔속도. 가중치 정규화 누락 주의.
