# P7 — Unscented Kalman Filter (무향 칼만 필터 · 추적)

🟩 **추적/센서퓨전** | **언어**: C++ (Eigen) | **난이도**: 중상

## 목표
EKF의 야코비안 선형화 대신 **시그마 포인트**로 비선형 분포를 근사. **CTRV**(등속·등각속도) 모델로 회전하는 물체까지 더 정확히 추적.

## 핵심 흐름
```
1. 시그마 포인트 생성   증강상태(노이즈 포함) 2n+1개
2. 예측                각 시그마 포인트를 CTRV 비선형 식에 통과 → 평균/공분산 재계산
3. 측정 예측           시그마 포인트를 측정공간으로 사상
4. 보정                칼만이득 K로 상태 갱신, NIS로 일관성 점검
```

## 빌드 / 데이터
- `reference/.../project_7_unscented_kalman_filter/` (`CMakeLists.txt`, `data/`)
- 스켈레톤은 `ukf.cpp` 핵심만. 레퍼런스 src에 끼워 빌드.

## 핵심 개념
- **CTRV 상태**: [px, py, v, ψ(yaw), ψ̇(yaw rate)]
- **시그마 포인트**: 평균±√((λ+n)P) 방향 샘플 → 비선형 통과 후 통계 복원.
- **NIS(정규화 혁신 제곱)**: 노이즈 파라미터 튜닝 일관성 검증(χ² 분포와 비교).
- 각도(ψ) 정규화 항상 주의.
